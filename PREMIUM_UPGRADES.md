# Premium Feel Upgrade Notes

This build adds six production-level upgrades focused on mobile ease of use and stronger infographic output quality.

## 1. Floating selected-block action bar
Selecting a block now shows a compact floating action bar for duplicate, layer back, layer front, and delete. This avoids forcing users to dig through the Edit panel for common object actions.

## 2. Magnetic alignment guides
Dragged blocks now snap to professional alignment targets: canvas margins, center lines, and major horizontal/vertical anchors. Visible guides appear only while dragging.

## 3. Presentation-style preview mode
The Export panel now includes Preview. It hides the editor chrome and shows the infographic on a dark presentation backdrop so the user can judge the final output before exporting.

## 4. Richer template thumbnails
Template cards now show mini layout previews instead of plain blank thumbnails, making template selection faster and more visual without adding long helper text.

## 5. Stronger Pro Polish behavior
The Polish button now improves sizing, spacing, z-order, logo strips, contact bars, stats, shapes, and dividers more intelligently. It is designed as a final cleanup pass before export.

## 6. More premium app and output styling
The app shell, canvas frame, cards, logo slots, selected-block states, and exported infographic details were refined with cleaner shadows, subtle borders, sheen, and card accents.
